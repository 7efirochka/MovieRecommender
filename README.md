# MovieRecommender

Cистема рекомендаций фильмов на основе контента (жанры + описание) с REST API.

## Описание

Сервис рекомендует фильмы, похожие на заданные пользователем названия, используя:
- **Жанровое сходство** (MultiLabelBinarizer + косинусное сходство)
- **Текстовое сходство описаний** (TF-IDF + косинусное сходство)
- **Гибридный подход** с весами 0.5 (жанры) и 0.5 (описание)

Система учитывает:
- Год выпуска (рекомендации ближе по времени)
- Рейтинг (высокорейтинговые фильмы в приоритете)
- Фильтрацию "взрослого" контента (комедии и семейные фильмы исключаются из категории "взрослых")

## Локальный запуск

Ниже представлены команды для локального запуска приложения
```bash
uv sync
uv run uvicorn src.api:app --reload --host 0.0.0.0 --port 8000

```

## Запуск с помощью Docker

```bash
docker build -t movie-recommender .
docker run -d -p 8000:8000 --name movie-api movie-recommender
```

Приложение будет работать на http://localhost:8000
